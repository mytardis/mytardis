Instructions for file access via SFTP and SSHFS

You can access all the files you have access to on the web via the SFTP protocol.

This allows you to use Desktop programs like Filezilla or Cyberduck to browse and download your files.

On some systems you can also mount MyTardis as a drive using SSHFS.
